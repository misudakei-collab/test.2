# COACHTECHフリマ（フリマアプリ）

メルカリのUIをベースに刷新した、本格的なフリマアプリです。クレジットカードとコンビニ決済のハイブリッド決済、スマートな検索機能、マイページ連動などを実装しています。

---

## 環境構築

### Dockerビルド
* `git clone <レポジトリのSSHまたはHTTPSのURL>`
* `docker-compose up -d --build`

### Laravel環境構築
* `docker-compose exec php bash`
* `composer install`
* `cp .env.example .env`（※環境変数を変更し、Stripe等の鍵を設定してください）
* `php artisan key:generate`
* `php artisan migrate`
* `php artisan db:seed`

---

## 開発環境
* **トップページ（商品一覧）** : http://localhost/
* **ユーザー登録** : http://localhost/register
* **phpMyAdmin** : http://localhost:8080/

---

## 主な実装機能
* **共通ヘッダー検索機能**（Enterキー連動、ブランド名部分一致拡張）
* **ハイブリッド決済機能**（Stripeクレジットカード決済 / コンビニ決済対応）
* **配送先住所の個別変更機能**（購入画面へのスマートリダイレクト）
* **マイページ機能**（出品した商品 / 購入した商品のタブ完全切り替え連動）
* **メルカリ風UIブラッシュアップ**（2カラム商品詳細、バッジ型チェックボックス、グリッド一覧）

---

## 仕様の変更・独自カスタマイズ点（重要）

元の基本仕様から、ユーザーの利便性を考慮して以下の変更・拡張を行っています。

1. **購入完了後の画面遷移の最適化**
   * 元の仕様には「購入完了後のサンクスページ」が存在しなかったため、決済完了後は自動的にデータベースを更新（`SOLD`状態へ変更）し、スムーズに元の画面や適切な導線へ戻る形に最適化しました。
2. **お気に入り（いいね）機能の仕様変更**
   * 「お気に入り登録した商品はおすすめタブから消えてお気に入りページへ移動する」という元仕様から、ユーザーが一覧で認知しやすいよう**「お気に入り登録後もおすすめタブに残し、ひと目で分かるようハートマークが点灯（付与）する」**仕様へと変更・機能強化しました。
